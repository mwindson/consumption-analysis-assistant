# 消费分析助手
A brand knowledge graph based on React & D3

## 技术框架

页面框架：React, React-Router

关系图：D3

数据管理：Redux-Saga

## 系统功能

+ 搜索品牌知识
+ 品牌、人物、商品的关系图
+ 品牌、人物、商品的详情卡片
